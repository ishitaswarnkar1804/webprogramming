# NEWS MAGAZINE PORTAL – Multi-Section Web Page

## Scenario
You are hired to design the homepage for an online news magazine **“Daily Insight”**.  
The page must be fully semantic and use all the mandatory HTML tags.

---

## Requirements

### Structure
- Use `<header>` for website logo and top navigation.
- Use `<nav>` containing internal links (`<a>`) to sections such as:
  - Politics  
  - Technology  
  - Sports  
  - World  
  - Editorial
- Use `<section>` to group the main news categories.
- Use **at least three `<article>` elements**, each containing:
  - Headline (`<h2>`)
  - Author name
  - Published date (using `<time>`)
  - A relevant image (`<img>` with proper `alt` text)
  - One to two paragraphs of content
- Add an `<aside>` on the right with:
  - A list of trending topics (ordered or unordered list)
  - An advertisement image
  - External resource links
- Use character references for symbols such as:
  - `&copy;`
  - `&rarr;`
  - `&nbsp;`

---

## Footer
- Use `<footer>` to display:
  - Copyright information
  - Contact links
  - Social media icons
