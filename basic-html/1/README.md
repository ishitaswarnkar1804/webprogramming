NEWS MAGAZINE PORTAL – Multi-Section Web Page
Scenario:
You are hired to design the homepage for an online news magazine “Daily Insight”. The page must be fully semantic and use all the mandatory HTML tags.
Requirements:
Structure:
•	Use <header> for website logo + top navigation.
•	Use <nav> containing internal links (<a>) to sections such as Politics, Technology, Sports, World, and Editorial.
•	Use <section> to group the main news categories
•	Use at least three <article> elements, each containing:
o	Headline (<h2>)
o	Author name
o	Published date (using <time>)
o	A relevant image (<img> with alt text)
o	1–2 paragraphs of content
•	Add an <aside> on the right with:
o	list of trending topics (ordered/unordered list)
o	advertisement image
o	external resource links
•	Use character references for symbols such as &copy;, &rarr;, &nbsp;.
Footer:
•	Use <footer> to display copyright, contact links, and social icons.
Deliverables:
File: news_portal.html
